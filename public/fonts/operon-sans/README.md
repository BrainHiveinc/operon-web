# Operon Sri - The Agent Font

**A distinctive geometric typeface for AI agents and autonomous systems.**

## About

Operon Sri is built on **Space Grotesk** - a modern geometric sans-serif with wide apertures and unique character forms. We've customized it with tighter letter spacing (-0.02em to -0.04em) and specific font weights to create a signature look for AI interfaces.

## What Makes It Different

- **Geometric Design**: Based on Space Grotesk's distinctive geometric letterforms
- **Tight Letter Spacing**: Custom -0.02em to -0.04em spacing for modern, compact look
- **Wide Apertures**: Enhanced readability at all sizes
- **Unique Characters**: Distinctive 'a', 'g', and numerals
- **Technical Precision**: Perfect for code, data, and technical content
- **JetBrains Mono**: For code blocks and technical displays

## Design Philosophy

Operon Sri represents:
- **Autonomy**: Independent, confident geometric letterforms
- **Intelligence**: Smart spacing and optical corrections
- **Precision**: Technical accuracy in every glyph
- **Modernity**: Contemporary geometric aesthetic
- **Scale**: Designed to perform from 10px to 200px

## Weights

- Light (300)
- Regular (400)
- Medium (500)
- SemiBold (600)
- Bold (700)

## Usage

```css
/* Import the font */
@import url('/fonts/operon-sans/operon-sans.css');

/* Use in your CSS */
font-family: 'Space Grotesk', 'Operon Sri', sans-serif;
letter-spacing: -0.02em;
```

## Technical Details

- **Base Font**: Space Grotesk (Google Fonts)
- **Monospace**: JetBrains Mono
- **Letter Spacing**: -0.02em (body), -0.04em (headings)
- **Font Weight**: 400 (body), 700 (headings)
- **Rendering**: Antialiased with optimizeLegibility
